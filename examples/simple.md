# Simple Document

This is a basic example of Markdown to PDF conversion with Pdfy.

## Getting Started

Converting this file is simple:

```bash
pdfy convert examples/simple.md
```

## Features Demonstrated

- **Basic text formatting**
- Headers and subheaders
- Code blocks
- Simple lists
